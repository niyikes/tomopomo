# PomoFlow

a clean, simple pomodoro timer i built to actually use myself!

## features
- focus / short break / long break modes that auto-switch
- add tasks with estimated pomodoros (decimals work!)
- click a task to set it as your active focus
- pomo dots show your progress toward the long break
- space bar to start/pause
- tab title updates with the countdown

## coming soon

- and whatever else seems useful

- tailwind css
- framer motion
