# tomopomo

a cute pomodoro timer i built to actually use myself!

## features
- focus / short break / long break modes
- add tasks with estimated pomodoros (decimals work)
- pomo dots that show progress toward the long break
- space bar to start/pause
- tab title updates with the clock countdown

- tailwind css
- framer motion
